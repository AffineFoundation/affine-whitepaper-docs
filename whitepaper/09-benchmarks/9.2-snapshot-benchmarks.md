# 9.2 Snapshot Benchmarks

**Directional signals only.** In contrast to the stable benchmarks in Section 9.1, these two additional benchmarks provide directional signals but should **not** be used for definitive ranking due to incomplete runs or uneven coverage.

The following table shows the snapshot results. The dashes and asterisks indicate where data is incomplete -- interpret with caution.

| Model | ToolSandbox Similarity | Tau2 pass_hat_1 |
|---|---|---|
| **Base (Qwen3-32B-TEE)** | 0.069 | 0.435* |
| **Axon1 M19** | 0.481 | 0.085 |
| **Leary CX** | 0.454 | 0.110 |
| **Leary CS** | -- | 0.131 |

> **Legend:** * = Base Tau2 score is from an earlier incomplete historical run and is not directly comparable to the formal miner runs. -- = data not available.

---

## ToolSandbox Results

ToolSandbox scores output similarity to reference trajectories across multi-tool agent interactions.

- Both Axon1 and Leary CX **dramatically outperform** the base (0.48/0.45 vs. 0.07).

> **Caveat:** Leary CS data is not available for this benchmark, so the comparison is incomplete.

## Tau2 Results

Tau2 measures end-to-end agentic task success.

- The base model's apparent advantage (0.435 vs. 0.085-0.131) reflects **incomparable run conditions** -- the base score comes from an earlier, incomplete run.

> **Key caveat:** This result should not be interpreted as a regression. The base and miner scores were collected under different conditions and are not directly comparable.
