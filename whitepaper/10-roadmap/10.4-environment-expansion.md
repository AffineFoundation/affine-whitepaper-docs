# 10.4 Environment Expansion

**Extensible by design.** The Affinetes framework supports arbitrary environment addition without changes to the scoring pipeline. Environment expansion is an ongoing process — the current suite covers software engineering, web interaction, memory management, tool-mediated planning, strategic reasoning, and distributional alignment, but there are significant capability axes not yet represented.

Directions under exploration include:

- **New problem domains** — extending beyond the current suite to cover areas such as scientific reasoning, multi-modal interaction, enterprise workflows, and other real-world agent scenarios where renewable task generation is feasible.
- **Deeper coverage of existing axes** — for example, longer-horizon software engineering tasks that require architectural reasoning, broader web domains beyond financial and weather data, and stronger game opponents that stress strategic reasoning more effectively.
- **Multi-agent coordination** — evaluating cooperative and competitive behaviors between multiple agents operating in shared environments.
- **Knowledge evaluation** — a multi-benchmark environment for factual accuracy and instruction-following is built and deployed but not yet included in the active scoring configuration.

> Each new environment broadens the capability surface that miners must cover to earn high geometric mean scores, creating compounding pressure toward general intelligence rather than narrow specialization.
