Affine is an open evaluation platform focused on high-quality RL environments. where open-source models by miner contributors are incentivized to outperform "existing ones" (@Wajahat please change this to a proper term. It refers to existing open-source models like Qwen) in real world agentic tasks. The platform empowers the community to break the grip of closed organizations in reasoning progress. The platform can also act as a public mechanism with the ability to verify and reward real incremental improvements. The key value of Affine lies in delivering RL environments and reward signals and helping miners through post training. It allows miners to define capability boundaries and enable them to steer improvement direction. to conduct open competition into a compounding (@Wajahat can replace a new word here) process instead of a single benchmark race. The platform's ecosystem generates robust and multidimensional environments where miners conduct RL post-training against the unified suite. In the next process, a multi-dimensional layer verifies and incentivizes the individual performance of each model using exploit-resistant mechanism. Deployed as Bittensor Subnet 120, the settlement pipeline — noise-adjusted Pareto filtering, cross-round ELO, geometric-mean aggregation, and floored normalization — allocates on-chain emissions in proportion to verified capability gains. A dual signal anti-copy layer that combines hidden state norms with top-$k$ logprob similarity, economically blocks replication, pushing for genuine training instead. The consistent initial evidence shows that these three independently trained models have outperformed the unmodified Qwen3-32B base on external benchmarks.
