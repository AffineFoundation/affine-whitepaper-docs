---
title: "4.4 The OpenEnv Protocol: A Standardized Training Interface"
---

**Multi-turn interaction.** Beyond single-turn evaluation, Affinetes defines **OpenEnv** — a standardized multi-turn interaction protocol that makes environments directly usable in reinforcement learning loops.

**Gym-like pattern.** The protocol follows a gym-like pattern:

```python
session = await env.openenv().reset(task_id=N, seed=S)
response = await session.step(action)       # returns observation, reward, done, info
await session.stop()                        # cleanup episode
```

Multiple episodes can run concurrently against the same environment instance.

**Lifecycle separation.** This interface separates two distinct concerns:

- **Episode lifecycle** — individual evaluation runs, each with independent state
- **Environment instance lifecycle** — the long-lived container that hosts many episodes

> **Why this matters:** For training loops that require thousands of episodes, this design eliminates the overhead of container creation per episode. An environment container persists across many episodes, amortizing startup cost.
